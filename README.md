 # Spotify Tracks Analysis

 Описание проекта
Проект посвящён анализу данных Spotify с использованием Python (pandas, matplotlib, seaborn) и PostgreSQL.  
Цель: очистка реального CSV-файла и проведение аналитических SQL-запросов для выявления закономерностей.

 Датасет
- track_id, track_name, track_number, track_popularity, explicit
- artist_name, artist_popularity, artist_followers, artist_genres
- album_id, album_name, album_release_date, album_total_tracks, album_type
- track_duration_min

Источник: Spotify (обработанный CSV)

 Этапы работы
1. Очистка CSV:
   - исправление кавычек и разделителей
   - обработка числовых данных
   - удаление битых строк
2. Загрузка данных в PostgreSQL
3. SQL-анализ:
   - Топ-10 треков и артистов
   - Влияние explicit-контента
   - Средняя длительность треков
   - Популярность по типу альбома
   - ABC-анализ артистов
4. Визуализация:
   - Распределение популярности треков
   - Топ жанров артистов
   - Корреляция длительности треков и популярности

 Используемые технологии
- Python: pandas, matplotlib, seaborn
- PostgreSQL
- pgAdmin
- CSV

 Результаты
- Чистый CSV для анализа
- SQL-запросы для аналитики
- Графики в `outputs/charts/`
- Выводы о популярных треках, артистах и жанрах

## ABC-анализ артистов по количеству фолловеров

Для оценки распределения влияния артистов был выполнен ABC-анализ
на основе суммарного количества подписчиков.

 Методология
1. Подсчитано общее количество фолловеров по каждому артисту
2. Артисты отсортированы по убыванию
3. Рассчитана накопленная доля (cumulative share)
4. Артисты разделены на классы:
   - A: до 80% аудитории
   - B: 80–95%
   - C: 95–100%

 Результаты
- Класс A включает небольшое количество артистов (например, Drake, Coldplay, Beyoncé),
  которые формируют основную часть аудитории Spotify
- Класс B - артисты со средним вкладом
- Класс C - длинный хвост артистов с минимальным влиянием

 Вывод
Аудитория Spotify характеризуется высокой концентрацией:
менее 10% артистов формируют более 80% всех подписчиков,
что подтверждает принцип Парето (80/20).
